HEXO命令指南

发表于 Jul 11 2016 | 次

Git Shell

cd/ 定位到c盘根目录

cd hexo 打开hexo文件位置，此时可进行以下hexo命令行：

hexo g == hexo generate     生成文档

hexo d == hexo deploy       部署到github

hexo s == hexo server       本地预览

hexo n == hexo new ""       新建博文

还能组合使用，如：

hexo d -g       生成并部署到github
赞赏也是刚需
